Tutorial and sample projects for Raspberry Pi

To help beginners to learn Linux programming and make DIY projects with the powerful Raspberry Pi®, we designed this learning kit which includes varies sensors, LCD display,servo, some supporting IC chips/devices.

We strongly believe the idea of "Learning by Doing". That is why we designed a step-by-step tutorial which comes with this learning kit. The tutorial has 21 lessons which evolve from a typical "Lighting LED" easy project to a complex lesson which uses Raspberry Pi to control multiple sensors and devices and make IOT home alarm system.

The tutorial provides and explains full example code for two kind of programming language ---- C language and Python language

If you have some basic C or Python programming experience, you will quickly understand Pi's I/O programming concept  through our kit and save a lot of research time to make your own embedded DIY project.

Even if you are a novice of C or Python language, you no need worry anything. Every lesson has detailed sample code,circuit graph and tutorial video. Just follow the step-by-step instruction and spend time to learn, you will enjoy the fun of doing DIY project and eventually become a master.
